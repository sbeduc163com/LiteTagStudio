# 蓝图标签打印工具 LiteTagStudio

蓝图标签打印工具是一款离线运行的 Windows 标签设计与打印工具，主要用于标签模板设计、Excel 数据导入、批量打印、扫码打印以及条码 / 二维码标签打印。

软件面向日常标签打印场景，适合用于商品标签、编号标签、条码标签、二维码标签、资产标签、图书标签、批量数据标签等打印需求。

## 主要功能

### 标签模板设计

支持自由设计标签内容，可在标签上添加：

- 文本
- 字段
- 图片
- 条形码
- 二维码
- 线条

标签尺寸、元素位置、宽高、边距等均使用毫米作为单位，便于按照实际标签纸尺寸进行设计，支持毫米级精度的标签排版。

### Excel 数据导入

支持导入 `.xls` 和 `.xlsx` 文件，将 Excel 表格数据用于标签打印。

导入后可以将 Excel 字段绑定到文本、条形码、二维码或图片元素中，实现批量生成不同内容的标签。

软件会按 Excel 单元格显示值读取数据，例如编号 `00000019` 会保留前导零。

### 批量打印标签

支持根据 Excel 数据批量打印标签。

每一行数据可以生成一张或多张标签，也可以结合打印设置中的打印份数进行重复打印。

适合连续打印大量编号、条码、二维码或其他数据标签。

### 扫码打印

支持在软件中直接输入内容，或使用扫码枪输入内容后按回车触发打印。

标签可以绑定内置的“自定义输入”字段，用于快速打印扫码内容、编号内容或临时输入内容。

### 条码和二维码

支持生成常见条形码和二维码标签。

条形码默认使用 Code128，适合多数编号、货号、批次号等打印场景。

### 打印预览

打印前可以进行预览，检查标签排版、分页、数据内容和打印位置，减少直接打印造成的浪费。

### 自定义纸张

支持在 Windows 系统中登记自定义纸张尺寸，方便适配不同规格的标签纸和条码打印机。

### 导出标签图片

支持在导入 Excel 数据后，将每个标签单独导出为 PNG 图片。

导出的对象是单个标签，不是整张纸，适合需要先生成标签图片再用于其他场景的情况。

### 离线运行

软件默认离线运行，不依赖服务器，不需要联网使用。

模板、配置和数据均保存在本机。

## 运行环境

- Windows 系统
- .NET Framework 4.8 或以上版本

如果系统未安装 .NET Framework 4.8，安装包会自动从微软官网下载并安装。
